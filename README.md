# Angular schematics Page Generator

Install by npm

    npm i page-generator

Create a new page

    ng g p path-of-page

The following component will be created

- \*.page.html
- \*.page.ts
- \*.page.css
- \*.module.ts
- \*.routing.ts
- \*.spec.ts

see the list of parameter [here](src/page-generator/schema.json)
